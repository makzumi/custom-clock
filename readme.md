# Customizable Clock

A highly customizable clock widget for KDE Plasma 6 that allows users to personalize time and date display with custom fonts, colors, and alignment options.

## Features

- **Time Format Options**: Switch between 12-hour and 24-hour format
- **Custom Fonts**: Choose from any installed font family for both time and date
- **Font Styling**: Customize font size, weight, and style independently
- **Color Customization**: Set custom colors for time and date elements
- **Flexible Alignment**: Adjust text alignment to suit your desktop layout

## Installation

1. Download the plasmoid package
2. Right-click on your desktop or panel
3. Select "Add Widgets..."
4. Click "Install from File..." and select the downloaded package
5. Add the Customizable Clock widget to your desktop or panel

## Configuration

Right-click on the widget and select "Configure Customizable Clock..." to access the settings.

## Requirements

- KDE Plasma 6
